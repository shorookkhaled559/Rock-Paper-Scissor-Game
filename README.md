# Rock Paper Scissor Game

This project is an **interactive Rock Paper Scissors game** that allows users to **play against the computer in real-time**. It provides a **fun, responsive, and visually appealing experience** entirely in the browser.

------------------------------------------------------------------------

## Live Demo

You can view the live version of the project here:  
**[Live Demo Link](https://shorookkhaled559.github.io/Rock-Paper-Scissor-Game/)**  

------------------------------------------------------------------------

## Features

- **Play Against Computer** --- Randomized computer choices.  
- **Score Tracking** --- Keeps track of user and computer scores.  
- **Interactive Icons** --- Rock, Paper, Scissors icons update dynamically.  
- **Responsive Design** --- Works perfectly on desktop, tablet, and mobile.  
- **Visual Feedback** --- Result highlights in colors: green for win, red for lose, gray for draw.  
- **Lightweight** --- Fully frontend, no backend required.  

------------------------------------------------------------------------

## Technologies Used

- **HTML5** --- Page structure and buttons.  
- **Tailwind CSS** --- Styling, layout, and responsive design.  
- **JavaScript (ES6)** --- Game logic, score updating, and icon animations.  
- **Font Awesome** --- Icons for Rock, Paper, and Scissors.

## How It Works

1. On page load:  
    - The game initializes scores to 0.  
    - User and computer icons are **hidden**.  
2. Users select **Rock, Paper, or Scissors**.  
3. Computer makes a **random choice**.  
4. Both selections are displayed as **icons and text**.  
5. Result is calculated: **Win, Lose, or Draw**.  
6. Scores are updated accordingly.  
7. The result box changes **color dynamically**: green for win, red for lose, gray for draw.  

------------------------------------------------------------------------

## Getting Started

1. **Clone or download** this repository.  
2. Open `index.html` directly in your browser.  
3. Click on any icon (Rock, Paper, or Scissors) to play.  
4. Scores and result will update automatically.  
5. Customize CSS or Tailwind classes for colors, icons, or layout if desired.
